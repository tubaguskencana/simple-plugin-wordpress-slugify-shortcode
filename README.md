# Installation
1. Download as zip 
2. Go to dashboard/ admin panel wordpress
3. Find menu 'plugin' and then 'add new' 
4. Find 'Upload Plugin' button
5. Choose downloaded file
6. Activate plugin
7. Put this shortcode anywhere you want **[form_slugify]**

# Feature
1. Copy indicator
2. User friendly form
3. Simple to use
